The experiment consists of four blocks that each have three parts.

1. Word pair learning

In the first part of each block you learn a list of word pairs.
Your task is to remember what words are associated with each other.
You also get the chance to practice the word pairs, by typing each word once given the other as a cue.
You will later (in the third part of each block) have to recall one of the words (the target) based on the other word (the cue).
Either word can later be the cue.
The number of word pairs you have to learn in each block varies from block to block.

2. Letter memory task

In the second part you will be performing a short letter memory task.
your task here is to recall a sequence of letters and type them in in the correct order once you are prompted to do so.
You have limited time to do so.

3. Word recall task

And finally, in the third part of each block you have to recall the words you learned in the first part of each block.

The words have to be recalled using one of the words in each word pair as a cue. So if you learned the word pair ruin - shape and we present you with the word ruin your task is to recall the word shape.
During the cue presentation we ask you to only press the spacebar once you have recalled the target word. Once you have done so, you have only a few seconds to type in the first three letters of the target word.
You can also type in the whole word, but only the first three letters have to be correct for you to get points.


If you really cannot recall a word, you still need to press the space bar to continue, but then please respond by typing 0 (zero) into the response field.


Each block is entirely seperate. This means, that we will never ask you about words from previous blocks.
You will be allowed to take a short break of max 1 minute between the blocks.
You will also receive feedback about your performance at the end of each block.


Accuracy is very imprortant for this experiment, but please also try to be as fast as possible, when trying to remember the target word!
